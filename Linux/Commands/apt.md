# apt

Purpose:
*Used to install and manage software packages on Debian-based Linux distributions like i have used Mint XFCE.*

Used In:

- Ubuntu
- Linux Mint
- Debian
- Pop!_OS

Syntax:

`apt <action> <package>`

Common Commands:

`sudo apt install git`

Install Git.

`sudo apt remove git`

Remove Git.

`sudo apt update`

Update package lists.

`sudo apt upgrade`

Upgrade installed packages.

`apt search python`

Search for packages.

Why Use It?

APT downloads software from official repositories and manages dependencies automatically.

Real World Example:

When I used Linux Mint, I installed software using:

`sudo apt install package-name`

Things to Remember:

- Fedora uses DNF.
- Linux Mint uses APT.
- Commands are similar but package managers are different.
