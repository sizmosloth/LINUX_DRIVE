# rm

Purpose:
*Used to remove files and directories*

Examples:

`rm index.html`
Deletes index.html in the current folder

`rm -r public`
Deletes public folder and everything inside it

`rm -f rm.md`
Force Deletes rm.md file without confirmation

`rm -rf public`
Force Deletes and recursively deletes all items in a folder and the folder that is public in this case
