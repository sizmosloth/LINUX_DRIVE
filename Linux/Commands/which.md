# which

Purpose:
*Used to locate where a command is installed*

Syntax:

`which <command>`

Examples:

`which python`

`which python3`

`which git`

`which bash`

Output:

`/usr/bin/python3`

This means Python is installed inside:

`/usr/bin`

Why Use It?

Useful for checking:

- Whether a program is installed.
- Which version of a program is being used.
- The location of an executable.

Things to Remember:

- `which` works for executable commands.
- It does not search every file on the system.

My Notes:

*I use `which` whenever I want to know where a command is located.*
