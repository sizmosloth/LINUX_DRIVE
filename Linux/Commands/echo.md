# echo

Purpose:
*Used to display text or the value of variables in the terminal.*

Syntax:

`echo <text>`

Examples:

`echo "Hello World"`

Output:

`Hello World`

Examples:

`echo "Learning Linux"`

Output:

`Learning Linux`

Environment Variable Example:

`echo $HOME`

Shows my Home Directory.

`echo $USER`

Shows my current username.

`echo $PATH`

Shows all directories where Linux searches for commands.

Real World Example:

While writing Bash scripts, I can use:

`echo "Program Started"`

to display messages.

Things to Remember:

- `echo` is one of the most commonly used commands.
- Useful in scripts.
- Useful for checking environment variables.

Practice:
