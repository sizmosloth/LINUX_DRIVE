# man

Purpose:
*Used to display the manual (Documentation) of linux commands*

Sytax:
`man <command>`

Examples:

`man ls`
Show the manual page for ls.

`man grep`
Show the manual page for grep.

`man chmod`
Show the manual page for chmod.

Guiding the Manual:
- Arrow keys to move lines up down
- Space for one page down
- / Search inside the manual
- q for quit

Why Use It?

Instead of searching Google every time, Linux already has documentation built into the system.

Real World Example:

I forgot how the `find` command works.

Instead of opening Chrome:

`man find`

Things to Remember:

- Almost every Linux command has a manual page.
- It explains the command, syntax and options.
- Press `q` anytime to quit.

My Notes:

*Whenever I forget a command, I should first check its manual using `man`.*
