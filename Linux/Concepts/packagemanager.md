# Package Manager

**Purpose:**
*Used to download, install, update and remove software.*

What is a Package Manager?

A package manager is a program that manages software on Linux.

Instead of downloading software manually from websites, Linux can install it directly from repositories.

**Examples:**

Fedora -> DNF

Linux Mint -> APT

Many Linux Distributions -> Flatpak

**How It Works:**

Repository -> Package Manager -> Application

**Examples:**

`sudo dnf install git`

`sudo apt install git`

`flatpak install flathub org.mozilla.firefox`

**Things to Remember:**

- Automatically installs dependencies.
- Keeps software updated.
- Downloads software from trusted sources.

Practice:

Search for a package using your package manager.
