# Filesystem

*Used to Store and Organize files and directories in Linux*

What is a Filesystem?
A Filesystem is a way linux stores and organises everything.

In Linux Everything starts from :

`/`

This is called **Root Directory**.

`/home` `/bin` `/usr` `/var` `/tmp` `/bin` all comes under `/`

**Think Like This:**

Imagine your computer is a city.

The filesystem is the map.

Folders are buildings.

Files are things inside those buildings.

Everything has its own location.

Real World Example:

My Connector project is located at:

`/home/sizmosloth/Documents/GitHub/Connector`

Linux follows this path to find my project.

Why is it Important?

Every command uses the filesystem.

Examples:

`cd`

Changes location.

`ls`

Lists files.

`pwd`

Shows my current location.

They all work because of the filesystem.

Things to Remember:

- Linux starts from `/`
- Every file has a location.
- Almost everything in Linux is treated as a file.

Summary:

Filesystem = The map of Linux.

Without it, Linux wouldn't know where any file exists.
