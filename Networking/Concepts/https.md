# HTTPS

Purpose:
*Used to securely transfer data between a client and a server.*

Examples:

`https://google.com`

`https://github.com`

Difference Between HTTP and HTTPS:

HTTP:
Data is not encrypted.

HTTPS:
Data is encrypted.

Why HTTPS Exists:

- Protect passwords
- Protect messages
- Protect payment information
- Improve security

My Notes:
*Most modern websites use HTTPS instead of HTTP.*

Real World Example:

When I login to GitHub, HTTPS helps protect my credentials while they travel across the internet.
