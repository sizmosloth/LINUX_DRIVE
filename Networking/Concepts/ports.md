# Ports

Purpose:
*Used to identify different services running on a computer.*

Common Ports:

`22`
SSH

`80`
HTTP

`443`
HTTPS

`3000`
Development Servers

`5432`
PostgreSQL

How It Works:

Computer
↓
Port 80 → Website

Computer
↓
Port 22 → SSH

My Notes:
*Think of ports as different doors of the same computer.*

Real World Example:

When I run Connector locally:

`localhost:3000`

3000 is the port being used by the application.
