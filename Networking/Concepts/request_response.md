# Request and Response

Purpose:
*Used for communication between a client and a server.*

What is a Request?

A request is sent by a client asking for information or an action.

Examples:

Browser - Request - Server

What is a Response?

A response is sent by the server back to the client.

Examples:

Server - Response - Browser

How It Works:

Client - Request - Server - Response - Client

My Notes:
*Almost everything on the internet works using requests and responses.*

Real World Example:

I open GitHub.

Browser - Request - GitHub Server - Response - GitHub Website Appears
