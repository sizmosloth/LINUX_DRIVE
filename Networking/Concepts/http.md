# HTTP

Purpose:
*Used to transfer data between a client and a server.*

Examples:

`http://example.com`

`http://localhost:3000`

How It Works:

Client (Browser)
↓
HTTP Request
↓
Server
↓
HTTP Response
↓
Client (Browser)

Common Request Methods:

`GET`
Get data from a server.

`POST`
Send data to a server.

`PUT`
Update existing data.

`DELETE`
Delete data.

My Notes:
*Whenever I open a website, my browser sends an HTTP request and receives a response from the server.*

Real World Example:

I open:

`http://localhost:3000`

My browser sends a request to my Connector app on and receives the webpage as a response.
