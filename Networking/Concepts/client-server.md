# Client and Server

Purpose:
*Used to understand how applications communicate.*

What is a Client?

A client is a device or application that requests data.

Examples:

- Browser
- Mobile App
- Desktop App

What is a Server?

A server is a computer or application that provides data.

Examples:

- Website Server
- Database Server
- API Server

How It Works:

Client - Request - Server - Response - Client

My Notes:
*Most applications work using the client-server model.*

Real World Example:

Connector:

Browser - Client

Connector Backend - Server - Database - Stores Data

When I send a message in Connector, the client sends a request to the server and the server stores or returns data.
