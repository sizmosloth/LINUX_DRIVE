# git clone

Purpose:
*git clone helps to copy a repo into our computer*

Examples:

`git clone https://github.com/sizmosloth/LINUX_DRIVE.git`
Clones LINUX_DRIVE into the current directory

`git clone https://github.com/sizmosloth/Connector.git`
Clones Connector also

*Helpful when i need to copy all the repo files from github to my computer*
