# git branch

Purpose:
*Used to create and manage branches.*

Examples:

`git branch`
Show all branches.

`git branch feature-login`
Create a new branch.

`git switch feature-login`
Switch to a branch.

`git switch main`
Switch back to main branch.

`git branch -d feature-login`
Delete a branch.
