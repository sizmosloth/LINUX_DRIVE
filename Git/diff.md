# git diff

**Purpose:**
*Used to show changes made to files before committing.*

**Syntax:**

`git diff`

**Examples:**

`git diff`

Show all changes.

`git diff filename`

Show changes in a specific file.

**Why Use It?**

Useful for checking what has changed before making a commit.

**Real World Example:**

I edited my `README.md`.

Before committing, I run:

`git diff`

to review my changes.

**Things to Remember:**

- Shows uncommitted changes.
- Helps avoid accidental commits.

**Practice:**

Create a file.

Edit it.

Run:

`git diff`
