# Git Basic Commands

Purpose:
*Git is a version control system used to track changes in code.*

`git init`
Initializes git in a folder

`git add file.txt` or `git add .`
Add a file to staging.

`git commit -m "message"`
Create a commit with a message.

`git status`
Show modified, staged, and untracked files.

`git log`
Show commit history.
