# mv

Purpose:
*Used to move or rename files and folders*

Examples:

`mv project.mp4 final_project.mp4`
Rename project.mp4 file to final_project.mp4

`mv project.mp4 Documents/Projects/`
Moves project.mp4 to the given path

`mv project.mp4 Documents/Projects/final_project.mp4`
Moves and renames at the same time

**You can check the results by ls command**

My notes:
*I use it for both moving and renaming files*
