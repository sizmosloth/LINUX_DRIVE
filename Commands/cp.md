# cp

Purpose:
*Used to Copy files and Folders*

Examples:

`cp notes.txt backup_notes.txt`
This creates a backup_notes.txt as a copy of notes.txt

`cp output.mp4 Videos/`
Copy output.mp4 file to Videos/ Dir

`cp -r LINUX_DRIVE Backup_LINUX_DRIVE`
Copies entire LINUX_DRIVE and makes another folder with name Backup_Linux_DRIVE
